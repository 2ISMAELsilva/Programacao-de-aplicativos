# comandos basicos

 1- adicionar arquivo para comit:
 git add . 

 2- Criar vercionamento (commit):
 git comit -m "comandos basicos de git"

3- Enviar alterações para o github:
 git push 

4- Trazendo possiveis alterações para o git:
git pull
